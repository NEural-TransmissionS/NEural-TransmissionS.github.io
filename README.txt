NETS Lab Website

This repository contains the source for the NEural TransmissionS (NETS) Lab website.

Structure

- `index.html`: homepage
- `about.html`: lab overview
- `research.html`: research, projects, and publications
- `people.html`: current team and alumni
- `join.html`: joining and collaboration information
- `contact.html`: contact information
- `assets/`: shared styles, scripts, and vendor files
- `images/`: branding, team photos, and site imagery

Deployment

This site is intended to be deployed from the repository root with GitHub Pages.
